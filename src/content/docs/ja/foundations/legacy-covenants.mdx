# YWAMのレガシー・ワードとカヴェナント

YWAMのアイデンティティと使命は、長年にわたり神様からローレン・カニングハムに与えられた「4つのレガシー・ワード（受け継ぐべき言葉）」によって明確にされています。これらは、宣教団体としての私たちの歩みを導き、形作ってきた重要な主の言葉です。

## 3. YWAM仙台へ与えられた言葉
これらは、特にYWAM仙台に対して与えられた主からの励ましと導きの言葉です。

1. **コミュニティを築く:** 愛と喜び、そしてリフレッシュ（回復）に満ちた場所。
2. **家族と若者:** 彼らに届き、回復をもたらすこと。
3. **神との親密さ:** 深い根、水のほとりに植わった木のような歩み。
4. **日本的キリスト教と文脈化:** 贖われた「祭り」、和楽器の使用など、日本の魂に響く表現。
5. **クリスチャンの団結:** 島々の架け橋となる（エペソ4:1-16、ヨハネ17:20-23）。
6. **再建と修復:** 仙台の街、そして傷ついた人々の回復。
7. **東北全県への開拓:** 東北6県のすべてに主の証しを立てる。
