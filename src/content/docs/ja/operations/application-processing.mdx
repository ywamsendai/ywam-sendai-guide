## 1. 初期トリアージ
新規申し込みの通知が届いたら：
* **内容確認:** 必須項目（推薦者情報を含む）がすべて入力されているか確認。
* **役割の特定:** 学生、スタッフ、チーム、訪問者のいずれであるかを確認。
* **Discord同期:** 内部管理システムの「Pending（保留中）」カテゴリに応募者を移動。

## 2. 推薦状の回収
* システムは自動的に推薦状依頼を送信します。
* 7日以上推薦状が届かない場合、**レジストラ**は応募者に手動でフォローアップの連絡をします。

## 3. 役割別のタスク

### 学生の場合:
1. 推薦状が揃い次第、スクール・リーダーに通知。
2. 承認後、**合格通知メール**と**デポジット（100,000円）の請求書**を送信。
3. 入金確認後、Discordのロールを「Registered Student」に変更。

### スタッフの場合:
1. 少なくとも2名のベースリーダーを含めた45分間のビデオ面接を調整。
2. 承認後、**スタッフ招待状（スタッフ採用通知）**を送信。
3. ビザスポンサーシップが必要な場合は、**ビザ担当者**に引き継ぐ。

### チーム・訪問者の場合:
1. 希望日程のハブの空き状況を確認。
2. 地域マップやハウスルールを含む**ビジター・パケット**を送信。

## 4. 記録管理
* 各スクールまたは年度の終了時に、すべての出願書類を「Completed（完了）」フォルダにアーカイブしてください。
