これらのプロトコルは、仙台ベースが**YWAMジャパン宗教法人**の全国基準に完全に準拠することを目的としています。すべての財務活動は、共同体の誠実さを保つために透明性をもって記録されなければなりません。

---

## 1. 会計サイクル
全国的な宗教法人としての法的義務を果たすため、年次報告の正確さは不可欠です。

* **会計年度：** 4月1日 〜 3月31日。
* **同期期限：** 毎年**4月15日**までに、ローカルデータを全国会計担当（後藤先生）に送付する必要があります。

---

## 2. ソフトウェア：会計王
ベースレベルのすべての入力には、全国標準ソフトウェアである**「会計王」**を使用します。

<Steps>
1.  **システム要件：** Windows 11に**日本語言語パック**がインストールされていること（データベースの正常な動作に必須）。
2.  **バックアップ：** 毎週バックアップを作成し、ベースの安全なドライブに保存してください。
3.  **データの整合性：** 取引を入力する前に、ローカルの拠点コードが「仙台」に正しく設定されていることを確認してください。
</Steps>

---

## 3. 勘定科目（収入）
すべての収入は、**「YWAMジャパン会計ハンドブック」**に従って分類されます。

| カテゴリ | 内容 |
| :--- | :--- |
| **受取寄付金** | 個人または団体からのすべての献金。**スタッフの月額献金（6,000円）およびゲストの宿泊費を含みます。** |
| **受取利息** | 銀行預金の利息。 |
| **雑収入** | その他のすべての収入。スクール（DTS/DBS）の学費を含みます。 |

---

## 4. 勘定科目（支出）
ベースのすべての支出に関する標準カテゴリです。

| カテゴリ | 内容 |
| :--- | :--- |
| **通信費** | ミニストリー関連の通信費（インターネット、オンラインサービスのサブスクリプション、郵便、電話など。ただしテレビ関連の費用は含まない）。 |
| **消耗品費** | 1点あたり10万円未満の消耗品（電子機器、家庭用品、その他少額の備品など）。 |
| **寄付金** | 団体、協会、学会、グループ等への寄付。受取人を明確に指定すること（個人への寄付は不可）。 |
| **備品購入費** | 1年以上使用し、1点あたり10万円以上の物品の購入。 |
| **交際費** | 季節の贈り物、葬儀の供花、お祝いなど、YWAM外の人々への贈り物（飲食代は含まない）。 |
| **保険料** | ミニストリーに関連する保険。 |
| **会議費** | 会議に関連する費用。一部の飲食代・軽食を含む（一人あたり5,000円以下）。摘要欄に会議参加者の氏名を記載すること。 |
| **諸会費・参加費** | ミニストリーに関連する会費（協会、クラブ等）やセミナーの参加費など。 |
| **伝道準備費** | ミニストリー活動の準備に関する一般経費。広告費、DTS等の講師を招くための交通費以外の経費を含む（DTSやノンスタッフとのミニストリーに伴う食費は可。スタッフのみの食費は不可）。 |
| **雑費** | 他のカテゴリに当てはまらない少額の経費。銀行手数料、登記簿謄本、印鑑証明書の費用を含む。また、全国事務所費を含むYWAMジャパン内の他の拠点への送金もここに含める。 |
| **事務用品費** | 用紙、筆記用具、コンピュータ周辺機器などの消耗事務用品。 |
| **給与・謝礼** | 講師、税理士、法的アドバイザーへの報酬、およびゲストスピーカー、通訳者等への謝礼。 |
| **預り源泉税** | 支払った報酬・謝礼に対する源泉徴収税。 |
| **賃借料** | 賃貸料またはリース料（レンタカーは含まない）。YWAM名義で借りている物件の住所を、その年度の最初の入力時に摘要欄に記載すること。 |
| **修繕費** | 維持管理および修理に関する費用（電球、粗大ごみ処分費用などを含む）。 |
| **旅費交通費** | ミニストリーに関連する旅行または交通費（「車両費」カテゴリに該当するものを除く）。 |
| **水道光熱費** | 電気、ガス、灯油、水道など（インターネットや電話は「通信費」となるため含まない）。 |
| **車両費** | ミニストリーに関連する車両使用の一般経費。ガソリン代、高速料金、駐車場代、レンタカー代を含む。 |

## 5. 特定のプロトコル
* **内部送金：** 他の拠点への送金は、二重課税を防ぐために「内部送金」として明確にタグ付けする必要があります。
* **領収書管理：** すべての支出には領収書（紙またはデジタル）が必要です。**5,000円**を超える購入については、事前にリーダーシップに相談してください。
* **源泉徴収：** 謝礼を支払う際は、源泉徴収を行う必要があります（日本居住のYWAMスタッフは3.063%、日本居住のノンスタッフは10.21%、海外居住者は一律20.42%）。これは宗教法人としての厳格な法的要件です。

<Aside type="note" title="財務の規律">
毎週定期的に入力することが、4月の負担を減らす鍵となります。特定の取引コードについて質問がある場合は、オフィスにある『会計ハンドブック』を参照してください。
</Aside>
