# 学費と諸費用

2026年度の各スクールにおける推定予算は以下の通りです。

---

## 1. スクール費用の内訳
DTSおよびセカンダリー・スクール（DBSなど）の基本費用は、学費と滞在費に分かれています。

<CardGrid>
  <Card title="授業料" icon="pencil">
    **300,000円** (税込)
    講義、事務手数料、リフレッシュメント（軽食等）を含みます。
  </Card>
  <Card title="滞在費" icon="home">
    **120,000円**
    寝具、光熱費、Wi-Fi利用料を含みます。
  </Card>
  <Card title="生活費の目安" icon="setting">
    **推定: 150,000円**
    食費（共同および個人分）、地域内の交通費、個人的な諸支出の目安です。
  </Card>
</CardGrid>

<Aside type="tip" title="コンボ割引">
DTSとDBSの両方に申し込む場合、DBSの授業料から **50,000円の割引** が適用されます（DTSを修了することが条件です）。
</Aside>

---

## 2. 伝道フェーズ (DTSのみ)
伝道（アウトリーチ）の費用は場所によって異なりますが、一般的に交通費、主要な食費、宿泊費が含まれます。
* **推定予算:** **400,000円**
* **注意:** 個人的な買い物や予防接種の費用は含まれません。

---

## 3. 支払スケジュールと方法
スクールの円滑な運営のため、以下のスケジュールを遵守してください。

| 項目 | 金額 | 期限 | 支払方法 |
| :--- | :--- | :--- | :--- |
| **予約金** | 100,000円 | 合格通知後 | 銀行振込 / Wise |
| **授業料残額** | 200,000円 | 到着の10日前まで | 銀行振込 / Wise |
| **滞在費** | 120,000円 | **到着時** | **現金のみ** |
| **伝道費（第1回）** | 150,000円 | 伝道出発の1ヶ月前 | **現金のみ** |

<Aside type="danger" title="重要：現金でのお支払い">
滞在費および伝道フェーズの費用は、**日本円（現金）での支払い**をお願いしています。到着前に、ご自身の銀行口座から現金を引き出せるよう準備を整えておいてください。
</Aside>

---

## 4. 返金ポリシー
* **授業料:** 支払い後の返金は原則として行われません。
* **滞在費・食費:** 週単位での日割り計算による返金が可能です。
* **伝道費:** 航空券のキャンセル料および、すでに拠点から支払い済みの返金不能な経費を除いた額を返金します。
