# 財務に関するお手続き

スクールの授業料、スタッフ費、または仙台での拠点購入に向けたご寄付など、皆様の尊いご支援に心より感謝いたします。

---

## 1. 日本国内からの送金
日本国内の銀行口座をお持ちの方は、ゆうちょ銀行への振込が最もスムーズです。

**ゆうちょ銀行 間での送金:**
* **記号:** 10140
* **番号:** 65305041
* **名義:** セカイセイネンセンキョウカイ

**他行からの振込:**
* **店名:** 〇一八（ゼロイチハチ）
* **店番:** 018
* **預金種目:** 普通預金
* **口座番号:** 6530504

---

## 2. 海外からの送金 (Wise)
海外から送金される場合は、手数料が最も安い **Wise** の利用を強くお勧めします。

<Aside type="danger" title="重要">
日本の銀行口座へ直接海外送金をすることはできません。必ずWiseのアプリまたはウェブサイトを経由して手続きを行ってください。
</Aside>

**受取人情報:**
* **受取人の種類:** 法人・団体 (Business / Charity)
* **口座名義:** セカイセイネンセンキョウカイ
* **銀行名:** [0310] GMOあおぞらネット銀行
* **支店名:** [101] 法人営業部
* **口座種別:** 普通
* **口座番号:** 2042380

## 4. 送金後のご連絡
入金を正確に確認するため、送金後は必ず **finance@ywamsendai.org** まで以下の内容をメールでお知らせください。
* 送金日
* 送金額
* 送金の目的（例：「DTS授業料」「スタッフ費」「建物基金への寄付」など）
