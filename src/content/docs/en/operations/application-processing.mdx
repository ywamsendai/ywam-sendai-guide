## 1. Initial Triage
When a notification for a new submission arrives:
* **Check Completeness:** Ensure all required fields (including references) are present.
* **Identify Persona:** Determine if the applicant is a Student, Staff, Team, or Visitor.
* **Discord Sync:** Move the applicant into the "Pending" category in our internal tracking system.

## 2. Reference Collection
* Our system triggers reference requests automatically.
* If a reference is delayed by more than 7 days, the **Registrar** must send a manual follow-up to the applicant.

## 3. Role-Specific Tasks

### For Students:
1. Notify School Leaders once references are in.
2. Upon approval, send the **Acceptance Email** + **Invoiced Deposit (¥100,000)**.
3. Once paid, move them to the "Registered Student" role on Discord.

### For Staff:
1. Schedule a 45-minute video interview with at least two base leaders.
2. Upon approval, send the **Staff Invitation Letter**.
3. Hand over the file to the **Visa Officer** if sponsorship is required.

### For Teams/Visitors:
1. Confirm Hub availability for the requested dates.
2. Send the **Visitor Packet** containing local maps and house rules.

## 4. Record Keeping
* Archive all applications in the "Completed" folder at the end of every school cycle or year.
